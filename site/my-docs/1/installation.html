Установка

Этот раздел содержит инструкцию по установке ПЕРСОНА МИС на персональный компьютер.

Перед началом установки

  • Убедитесь, что компьютер соответсвует минимальным системным требованиям.

  • Загрузите и распокуйте архив с файлами программы.

Установите ПЕРСОНА МИС

  1. В распокованном архиве запустите файл setup.exe.

  2. В открывшемся окне выберите путь для установки программы.

  3. Нажмите кнопку Далее.

  4. Дождитесь окончания установки.

Готово! Программа установлена на компьютер.
