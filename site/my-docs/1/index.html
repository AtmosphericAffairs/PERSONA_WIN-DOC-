О программе

ПЕРСОНА МИС — программный комплекс для обработки режимной метеорологической информации версии 2021.01. Он предназначен для специалистов центров и управлений по гидрометеорологии и мониторингу загрязнений окружающей среды.

Функции

ПЕРСОНА МИС выполняет функции:

  • сбора и хранения метеорологических данных;

  • контроля качества наблюдений;

  • формирования сводных ежемесячных таблиц.
